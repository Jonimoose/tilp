2. Overview

When you launch TiLP at once, you will see this :



In TiLP-II, you will notice there is not any menu bar any more. All operations/settings can be done by Drag & Drop or by popup menus (right click on each view).

By default, the window is splitted into 2 parts :
- the local view (right) : contains a toolbar (calculator related operations), a tree and an information frame (current folder),
- the remote view (left) : contains a toolbar (computer related operations), a list and an information frame (memory used and/or free).

This is the same behaviour as TiLP. but it's possible to configure TiLP-II to run with only the remote view. This will be explained later.

Any operation is based on the content of the left or the right frame.
